    基本

    ブラケット記法の基本

    問題

    量子力学において，状態ベクトルはディラックのブラケット記法を用いて扱われる。量子状態を表すベクトルをケット \(|\psi\rangle\)，その随伴に相当する線形写像をブラ \(\langle\psi|\) と呼ぶ。これらを用いることで，内積や外積，および演算子の作用を簡潔に表現できる。

    ある二次元量子系の正規直交基底を \(\{|0\rangle, |1\rangle\}\) とする。この基底は \(\langle 0|0\rangle = \langle 1|1\rangle = 1\)，\(\langle 0|1\rangle = \langle 1|0\rangle = 0\) を満たす。この系における2つの状態ベクトルが以下のように与えられている。

    \[ |\psi\rangle = \frac{1}{\sqrt{5}} |0\rangle + \frac{2}{\sqrt{5}} |1\rangle \]
    \[ |\phi\rangle = \frac{1}{\sqrt{2}} |0\rangle + \frac{i}{\sqrt{2}} |1\rangle \]

    以下の問いに答えよ。

    (1) 状態 \(|\psi\rangle\) および \(|\phi\rangle\) に対応するブラベクトル \(\langle\psi|\) と \(\langle\phi|\) を，ブラの基底 \(\{\langle 0|, \langle 1|\}\) を用いてそれぞれ書き下せ。

    (2) \(\langle\phi|\psi\rangle\) と \(\langle\psi|\phi\rangle\) を求め，\((\langle\phi|\psi\rangle)^*=\langle\psi|\phi\rangle\) を示せ。

    (3) 2つの外積から作られる演算子 \(\hat{A} = |\phi\rangle\langle\psi|\) について，これに右からケット \(|\psi\rangle\) を作用させた式 \(\hat{A}|\psi\rangle\) を計算し，結果を単一のケットベクトルとして表せ。

    解説

    (1)

    ケットベクトルからブラベクトルを作る際，各基底ベクトルは対応するブラ基底に置き換わり，その係数は複素共役になります。

    • \(\langle\psi|\) の導出 \(|\psi\rangle\) の係数はどちらも実数であるため，係数は変化しません。 $$ \langle\psi| = \frac{1}{\sqrt{5}} \langle 0| + \frac{2}{\sqrt{5}} \langle 1| $$

    • \(\langle\phi|\) の導出 \(|1\rangle\) の係数である虚数単位 \(i\) の符号が反転して \(-i\) となります。 $$ \langle\phi| = \frac{1}{\sqrt{2}} \langle 0| - \frac{i}{\sqrt{2}} \langle 1| $$

    (2)

    (1) で求めた \(\langle\phi|\) と，与えられた \(|\psi\rangle\) の内積を計算します。分配法則を用いて展開します。

    \[ \langle\phi|\psi\rangle = \left( \frac{1}{\sqrt{2}} \langle 0| - \frac{i}{\sqrt{2}} \langle 1| \right) \left( \frac{1}{\sqrt{5}} |0\rangle + \frac{2}{\sqrt{5}} |1\rangle \right) \]
    \[ = \frac{1}{\sqrt{10}} \langle 0|0\rangle + \frac{2}{\sqrt{10}} \langle 0|1\rangle - \frac{i}{\sqrt{10}} \langle 1|0\rangle - \frac{2i}{\sqrt{10}} \langle 1|1\rangle \]

    基底の正規直交条件 \(\langle 0|0\rangle = \langle 1|1\rangle = 1\)，\(\langle 0|1\rangle = \langle 1|0\rangle = 0\) を代入します。

    \[ \langle\phi|\psi\rangle = \frac{1}{\sqrt{10}} - \frac{2i}{\sqrt{10}} = \frac{1 - 2i}{\sqrt{10}} \]

    この結果の複素共役をとります。虚数部分の符号が反転します。

    \[ (\langle\phi|\psi\rangle)^* = \frac{1 + 2i}{\sqrt{10}} \]

    一方で，順序を入れ替えた内積 \(\langle\psi|\phi\rangle\) を同様に計算します。

    \[ \langle\psi|\phi\rangle = \left( \frac{1}{\sqrt{5}} \langle 0| + \frac{2}{\sqrt{5}} \langle 1| \right) \left( \frac{1}{\sqrt{2}} |0\rangle + \frac{i}{\sqrt{2}} |1\rangle \right) \]
    \[ = \frac{1}{\sqrt{10}} \langle 0|0\rangle + \frac{i}{\sqrt{10}} \langle 0|1\rangle + \frac{2}{\sqrt{10}} \langle 1|0\rangle + \frac{2i}{\sqrt{10}} \langle 1|1\rangle = \frac{1 + 2i}{\sqrt{10}} \]

    両者の結果が一致するため， \((\langle\phi|\psi\rangle)^* = \langle\psi|\phi\rangle\) が成り立つことが確認できました。

    (3)

    演算子 \(\hat{A} = |\phi\rangle\langle\psi|\) に右からケット \(|\psi\rangle\) を掛けます。ブラケット記法では，中央の結合部分が内積としてまとめられます。

    \[ \hat{A}|\psi\rangle = \big( |\phi\rangle\langle\psi| \big) |\psi\rangle = |\phi\rangle \langle\psi|\psi\rangle \]

    ここで，状態 \(|\psi\rangle\) は規格化されているため，自身との内積は 1 になります。実際に計算して確かめます。

    \[ \langle\psi|\psi\rangle = \left( \frac{1}{\sqrt{5}} \langle 0| + \frac{2}{\sqrt{5}} \langle 1| \right) \left( \frac{1}{\sqrt{5}} |0\rangle + \frac{2}{\sqrt{5}} |1\rangle \right) = \frac{1}{5} + \frac{4}{5} = 1 \]

    したがって，内積部分は \(\langle\psi|\psi\rangle = 1\) となります。これをもとの式に戻します。

    \[ \hat{A}|\psi\rangle = |\phi\rangle \cdot 1 = |\phi\rangle = \frac{1}{\sqrt{2}} |0\rangle + \frac{i}{\sqrt{2}} |1\rangle \]

    演算子 \(\hat{A}\) は，任意のベクトルを状態 \(|\phi\rangle\) の方向へと射影・変換するはたらきを持っていることがわかります。
