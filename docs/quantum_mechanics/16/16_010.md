# フォック空間と粒子数表示

## 問題
粒子数が一定ではなく，生成や消滅によって変動しうる多粒子系を扱うため，異なる粒子数を持つヒルベルト空間の直和としてフォック空間を導入する。いま，互いに直交する一粒子状態の全体系を $| 1 \rangle, | 2 \rangle, \dots, | k \rangle, \dots$ とする。各状態に入っている粒子数を $n_1, n_2, \dots, n_k, \dots$ と表すとき，フォック空間における状態ベクトルは粒子数状態 $| n_1, n_2, \dots, n_k, \dots \rangle$ を基底として展開される。以下の問いに答えよ。

(1) 粒子が全く存在しない状態である真空状態を $| 0 \rangle$ とする。真空状態におけるすべての状態の粒子数 $n_k$ を答えよ。また，全粒子数演算子を $\hat{N}$ とするとき，真空状態に対する固有値方程式 $\hat{N} | 0 \rangle = N_0 | 0 \rangle$ の固有値 $N_0$ の数値を答えよ。

(2) 1つの状態に任意の数の粒子が入りうるボソン系を考える。一粒子状態 $| 1 \rangle$ に 2 個，状態 $| 3 \rangle$ に 1 個の粒子が存在し，他の状態には粒子が存在しない多ボソン状態を粒子数表示の記号で書き表せ。また，この状態に全粒子数演算子 $\hat{N}$ を作用させたときの固有値を求めよ。

(3) パウリの排他原理が成り立つフェルミオン系を考える。ボソン系とは異なり，各状態の粒子数 $n_k$ がとりうる値の範囲に制限が生じる。このとき，各 $n_k$ がとりうるすべての数値を個別に列挙せよ。

---

## 解説

### (1)

真空状態とは，系を構成するいかなる一粒子状態にも粒子が 1 個も割り当てられていない空の状態を意味します。したがって，すべての状態添字 $k$ に対して，粒子数は次のようになります。

$$
n_k = 0
$$

全粒子数演算子 $\hat{H}$ は，各状態の粒子数 $n_k$ の総和を固有値として返す演算子です。真空状態 $| 0 \rangle$ はすべての $n_k$ が 0 であるため，その総和としての全粒子数も 0 と定まります。したがって，固有値方程式における固有値は次の数値になります。

$$
N_0 = 0
$$

### (2)

粒子数表示の基底 $| n_1, n_2, n_3, \dots \rangle$ の各位置に，問題文で指定された粒子の個数をそのまま当てはめます。状態 1 の位置に 2，状態 3 の位置に 1 を配置し，それ以外の成分をすべて 0 と置くことで，求める状態ベクトルは次のように表現されます。

$$
| 2, 0, 1, 0, 0, \dots \rangle
$$

全粒子数演算子 $\hat{N}$ をこの状態に作用させると，各一粒子状態に占有されている粒子数の総和が固有値として得られます。

$$
\hat{N} | 2, 0, 1, 0, 0, \dots \rangle = (2 + 0 + 1 + 0 + 0 + \dots) | 2, 0, 1, 0, 0, \dots \rangle
$$

括弧内の足し算を実行することにより，求める固有値は次のようになります。

$$
N = 3
$$

### (3)

フェルミオン系においては，同一の量子状態に複数の粒子が同時に進入することがパウリの排他原理によって禁止されます。

1つの状態に粒子が全く入っていないか，あるいは高々1個の粒子しか入ることができないという物理的制約から，任意の量子状態添字 $k$ における粒子数 $n_k$ がとりうる数値は，次の2つの値に限定されます。

$$
n_k = 0, 1
$$
