# 水素原子の隠れた対称性と縮退の計算

## 問題
水素原子のエネルギー準位は主量子数 $n$ のみによって決定され，軌道角運動量量子数 $l$ が異なる状態の間にも完全な縮退が存在する。この異常縮退は，空間の回転対称性に加え，クーロン場特有の保存量であるルンゲ・レンツベクトルが存在することに起因する。ここでは，ルンゲ・レンツ演算子と角運動量演算子の交換関係から状態間の遷移を具体的に計算し， $n=2$ における縮退の仕組みを解析する。

水素原子のハミルトニアンを $\hat{H}$ とし，軌道角運動量演算子を $\hat{\boldsymbol{L}}$ とする。ハミルトニアンが負のエネルギー固有値 $E$ を持つ束縛状態の空間において，修正されたルンゲ・レンツ演算子 $\hat{\boldsymbol{M}}$ を導入する。これらはハミルトニアンと可換であり，さらに角運動量演算子との間に以下の交換関係を満たす。

$$
[\hat{L}_i, \, \hat{M}_j] = i\hbar \sum_{k} \epsilon_{ijk} \hat{M}_k
$$

$$
[\hat{M}_i, \, \hat{M}_j] = i\hbar \left( -\frac{2E}{m} \right) \sum_{k} \epsilon_{ijk} \hat{L}_k
$$

ここで $\epsilon_{ijk}$ は完全反対称テンソルである。定数係数を整理するため，新しく演算子 $\hat{\boldsymbol{K}} = \sqrt{-\frac{m}{2E}}\hat{\boldsymbol{M}}$ を定義すると，交換関係は以下のように整理される。

$$
[\hat{L}_x, \, \hat{K}_y] = i\hbar \hat{K}_z, \quad [\hat{K}_x, \, \hat{K}_y] = i\hbar \hat{L}_z
$$

主量子数 $n=2$ のエネルギー準位に属する4つの状態は，軌道角運動量の基底 $|l, m\rangle$ を用いて $|0, 0\rangle$ ， $|1, 1\rangle$ ， $|1, 0\rangle$ ， $|1, -1\rangle$ と表される。これらはすべて共通のエネルギー固有値 $E$ を持つ。以下の問いに答えよ。

(1) 演算子 $\hat{K}_z$ を用いた交換関係 $[\hat{L}_z, \, \hat{K}_z]$ の値を計算せよ。また，この結果に基づき，状態 $|1, 0\rangle$ に $\hat{K}_z$ を作用させて得られる新しい状態 $\hat{K}_z |1, 0\rangle$ の $\hat{L}_z$ に対する固有値を求めよ。

(2) 演算子のパリティ対称性から， $\hat{K}_z |1, 0\rangle$ は $l=0$ の状態と結びつく。 $\hat{\boldsymbol{L}}^2$ と $\hat{\boldsymbol{K}}$ の交換関係から導かれる以下の具体的な行列要素の関係式を考える。

$$
\langle 0, 0 | \hat{K}_z | 1, 0 \rangle = \alpha
$$

ここで $\alpha$ は非ゼロの実数定数である。このとき，エルミート共役の関係を用いて行列要素 $\langle 1, 0 | \hat{K}_z | 0, 0 \rangle$ を $\alpha$ を用いて表せ。

(3) 状態 $|0, 0\rangle$ と $|1, 0\rangle$ の 2つの状態がなす部分空間において，演算子 $\hat{K}_z$ の行列表現を $\alpha$ を用いて 2行2列の行列で書き下せ。さらに，この行列の固有値方程式を解くことにより， $\hat{K}_z$ の固有値と，それに対応する規格化された固有状態を $\alpha$ ， $|0, 0\rangle$ ， $|1, 0\rangle$ を用いて具体的に求めよ。

---

## 解説

### (1)

交換関係 $[\hat{L}_z, \, \hat{K}_z]$ を計算します。与えられた交換関係において，同一の成分同士の交換子は 0 になります。

$$
[\hat{L}_z, \, \hat{K}_z] = \hat{L}_z \hat{K}_z - \hat{K}_z \hat{L}_z = 0
$$

この可換性を用いて，状態 $\hat{K}_z |1, 0\rangle$ に演算子 $\hat{L}_z$ を作用させます。状態 $|1, 0\rangle$ は $\hat{L}_z$ の固有値 0 の固有状態であるため， $\hat{L}_z |1, 0\rangle = 0$ です。

$$
\hat{L}_z \left( \hat{K}_z |1, 0\rangle \right) = \hat{K}_z \hat{L}_z |1, 0\rangle = \hat{K}_z \left( 0 \right) = 0
$$

したがって，新しい状態 $\hat{K}_z |1, 0\rangle$ もまた， $\hat{L}_z$ に対する固有値が 0 となる固有状態であることが求まります。

### (2)

ルンゲ・レンツ演算子およびそれから誘導される演算子 $\hat{\boldsymbol{K}}$ は，物理的にエルミート演算子（自己共役演算子）です。したがって， $\hat{K}_z^\dagger = \hat{K}_z$ が成り立ちます。

行列要素の複素共役の定義を適用します。

$$
\langle 1, 0 | \hat{K}_z | 0, 0 \rangle = \langle 0, 0 | \hat{K}_z^\dagger | 1, 0 \rangle^* = \langle 0, 0 | \hat{K}_z | 1, 0 \rangle^*
$$

定数 $\alpha$ は実数であるため，複素共役をとっても値は変化しません。

$$
\langle 0, 0 | \hat{K}_z | 1, 0 \rangle^* = \alpha^* = \alpha
$$

これより，求める行列要素は $\langle 1, 0 | \hat{K}_z | 0, 0 \rangle = \alpha$ となります。

### (3)

基底ベクトルを最初を $|0, 0\rangle$ ，2番目を $|1, 0\rangle$ とする順序で並べ，演算子 $\hat{K}_z$ の行列要素を決定します。
* 1行1列目： $\langle 0, 0 | \hat{K}_z | 0, 0 \rangle = 0$ （パリティ奇の演算子による同パリティ状態間の遷移のため 0 ）
* 2行2列目： $\langle 1, 0 | \hat{K}_z | 1, 0 \rangle = 0$ （同様の理由により 0 ）
* 1行2列目： $\langle 0, 0 | \hat{K}_z | 1, 0 \rangle = \alpha$
* 2行1列目： $\langle 1, 0 | \hat{K}_z | 0, 0 \rangle = \alpha$

これらを行列形式にまとめます。

$$
\hat{K}_z = \begin{pmatrix} 0 & \alpha \\ \alpha & 0 \end{pmatrix}
$$

この行列の固有値を $\lambda$ と置き，特性方程式を計算します。

$$
\begin{vmatrix} -\lambda & \alpha \\ \alpha & -\lambda \end{vmatrix} = 0 \implies \lambda^2 - \alpha^2 = 0 \implies \lambda = \pm \alpha
$$

したがって，固有値は $\lambda = \alpha$ および $\lambda = -\alpha$ です。

それぞれの固有値に対応する固有ベクトル $\begin{pmatrix} c_1 \\ c_2 \end{pmatrix}$ を求めます。

* **固有値 $\lambda = \alpha$ のとき**
  
$$
  \begin{pmatrix} 0 & \alpha \\ \alpha & 0 \end{pmatrix} \begin{pmatrix} c_1 \\ c_2 \end{pmatrix} = \alpha \begin{pmatrix} c_1 \\ c_2 \end{pmatrix} \implies \alpha c_2 = \alpha c_1 \implies c_1 = c_2
$$
  
  規格化条件 $2|c_1|^2 = 1$ より，固有状態は以下のようになります。
  
$$
  |\phi_+\rangle = \frac{1}{\sqrt{2}} |0, 0\rangle + \frac{1}{\sqrt{2}} |1, 0\rangle
$$

* **固有値 $\lambda = -\alpha$ のとき**
  
$$
  \begin{pmatrix} 0 & \alpha \\ \alpha & 0 \end{pmatrix} \begin{pmatrix} c_1 \\ c_2 \end{pmatrix} = -\alpha \begin{pmatrix} c_1 \\ c_2 \end{pmatrix} \implies \alpha c_2 = - \alpha c_1 \implies c_2 = -c_1
$$
  
  同様に規格化を行い，固有状態は以下のようになります。
  
$$
  |\phi_-\rangle = \frac{1}{\sqrt{2}} |0, 0\rangle - \frac{1}{\sqrt{2}} |1, 0\rangle
$$

この計算結果は，ハミルトニアンと可換な演算子 $\hat{K}_z$ が，異なる軌道角運動量量子数 $l=0$ と $l=1$ の状態を互いに混ぜ合わせる固有状態を形成することを示しています。角運動量を変化させる保存量の存在により，本来は異なるはずの $l$ の間でエネルギー準位が完全に一致する異常縮退が引き起こされる仕組みが，具体的な行列計算によって導かれました。
