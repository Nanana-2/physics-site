# 角運動量の合成

## 問題
ふたつの独立な角運動量が存在するとき，それらを足し合わせた全体の角運動量を全角運動量と呼ぶ。個々の角運動量の方向が別々に確定している状態を基底とする表現から，全角運動量の大きさとその $z$ 成分が確定している状態へと基底を変換する。この変換における展開係数はクレブシュ・ゴルダン（CG）係数と呼ばれ，角運動量の保存則に基づいた組み合わせから決定される。

ふたつのシステムにおいて，それぞれの角運動量を $\boldsymbol{J}_1$ と $\boldsymbol{J}_2$ とし，全角運動量を $\boldsymbol{J} = \boldsymbol{J}_1 + \boldsymbol{J}_2$ とする。それぞれの角運動量の大きさを指定する量子数を $j_1 = \frac{1}{2}$ ， $j_2 = \frac{1}{2}$ とする。それぞれの $z$ 成分量子数は $m_1 = \pm\frac{1}{2}$ ， $m_2 = \pm\frac{1}{2}$ の値をとる。

個々の成分が確定した直積基底を $|m_1, m_2\rangle$ と表し，全角運動量の同時固有状態を $|J, M\rangle$ と表す。全角運動量の大きさがとり得る範囲は $|j_1 - j_2| \le J \le j_1 + j_2$ であり， $z$ 成分量子数の間には $M = m_1 + m_2$ の関係が成り立つ。以下の問いに答えよ。

(1) 全角運動量量子数 $J$ がとり得るふたつの値を示せ。また，それぞれの $J$ の値に対して， $z$ 成分量子数 $M$ がとり得る値をすべて挙げよ。

(2) $M$ の最大値に対応する全角運動量の状態は，直積基底の最大値の組み合わせと一対一で対応し， $|1, 1\rangle = |+\frac{1}{2}, +\frac{1}{2}\rangle$ と表される。この状態の両辺に全消滅演算子 $\hat{J}_- = \hat{J}_{1-} + \hat{J}_{2-}$ を作用させることで，状態 $|1, 0\rangle$ を直積基底の重ね合わせとして表せ。ただし，演算子の作用規則は $\hat{J}_- |J, M\rangle = \sqrt{J(J+1) - M(M-1)}\hbar |J, M-1\rangle$ とする。

(3) 状態 $|0, 0\rangle$ は， $M = m_1 + m_2 = 0$ となる直積基底の組み合わせから構成され，かつ (2) で求めた状態 $|1, 0\rangle$ と互いに直交する。この条件と規格化条件から，状態 $|0, 0\rangle$ の具体的な展開式を求めよ。ただし， $|+\frac{1}{2}, -\frac{1}{2}\rangle$ の係数は正の実数とせよ。

---

## 解説

### (1)

全角運動量量子数 $J$ がとり得る範囲の公式に， $j_1 = \frac{1}{2}$ と $j_2 = \frac{1}{2}$ を代入します。

* 最小値： $| \frac{1}{2} - \frac{1}{2} | = 0$
* 最大値： $\frac{1}{2} + \frac{1}{2} = 1$

したがって， $J$ がとり得る値は $J = 1$ および $J = 0$ のふたつです。

それぞれの $J$ に対応する $M$ の範囲は $-J \le M \le J$ であるため，以下のようになります。

* $J = 1$ のとき： $M = 1, 0, -1$
* $J = 0$ のとき： $M = 0$

全体の状態で数えると， $3 + 1 = 4$ つの状態が存在します。これは元の直積基底の数である $2 \times 2 = 4$ と一致します。

### (2)

最大の状態 $|1, 1\rangle = |+\frac{1}{2}, +\frac{1}{2}\rangle$ の両辺に消滅演算子を作用させます。全体の定数因子である $\hbar$ はあらかじめ省略して計算します。

* **左辺の計算**
  
  $J = 1, M = 1$ を作用規則の根号の式に代入します。
  
$$
  \hat{J}_- |1, 1\rangle = \sqrt{1(1+1) - 1(1-1)} |1, 0\rangle = \sqrt{2 - 0} |1, 0\rangle = \sqrt{2} |1, 0\rangle
$$

* **右辺の計算**
  
  全消滅演算子はそれぞれの空間に独立に作用するため， $\hat{J}_- = \hat{J}_{1-} + \hat{J}_{2-}$ を分配します。
$$
  (\hat{J}_{1-} + \hat{J}_{2-}) |+\frac{1}{2}, +\frac{1}{2}\rangle = \left( \hat{J}_{1-} |+\frac{1}{2}\rangle \right) |+\frac{1}{2}\rangle + |+\frac{1}{2}\rangle \left( \hat{J}_{2-} |+\frac{1}{2}\rangle \right)
$$
  
  それぞれの成分において $j = \frac{1}{2}, m = \frac{1}{2}$ を作用規則に代入すると，係数は $\sqrt{\frac{1}{2}(\frac{1}{2}+1) - \frac{1}{2}(\frac{1}{2}-1)} = \sqrt{\frac{3}{4} + \frac{1}{4}} = 1$ となります。

$$
  = |-\frac{1}{2}, +\frac{1}{2}\rangle + |+\frac{1}{2}, -\frac{1}{2}\rangle
$$

左右の両辺を等号で結び，全体を $\sqrt{2}$ で割り算します。

$$
\sqrt{2} |1, 0\rangle = |-\frac{1}{2}, +\frac{1}{2}\rangle + |+\frac{1}{2}, -\frac{1}{2}\rangle
$$

$$
|1, 0\rangle = \frac{1}{\sqrt{2}} |-\frac{1}{2}, +\frac{1}{2}\rangle + \frac{1}{\sqrt{2}} |+\frac{1}{2}, -\frac{1}{2}\rangle
$$

これが求める状態の展開式です。ふたつの直積基底に対する CG 係数は，ともに $\frac{1}{\sqrt{2}}$ になります。

### (3)

状態 $|0, 0\rangle$ の $z$成分量子数は $M = 0$ であるため，関係式 $m_1 + m_2 = 0$ を満たす直積基底だけで構成されます。該当する基底は $|+\frac{1}{2}, -\frac{1}{2}\rangle$ と $|-\frac{1}{2}, +\frac{1}{2}\rangle$ のふたつです。

未知の係数を $A, B$ と置いて状態を仮定します。

$$
|0, 0\rangle = A |+\frac{1}{2}, -\frac{1}{2}\rangle + B |-\frac{1}{2}, +\frac{1}{2}\rangle
$$

(2) で求めた状態 $|1, 0\rangle$ との直交条件を計算します。直積基底同士は互いに正規直交します。

$$
\langle 1, 0|0, 0\rangle = \left( \frac{1}{\sqrt{2}} \langle -\frac{1}{2}, +\frac{1}{2}| + \frac{1}{\sqrt{2}} \langle +\frac{1}{2}, -\frac{1}{2}| \right) \left( A |+\frac{1}{2}, -\frac{1}{2}\rangle + B |-\frac{1}{2}, +\frac{1}{2}\rangle \right) = 0
$$

$$
\frac{1}{\sqrt{2}} B + \frac{1}{\sqrt{2}} A = 0 \implies B = -A
$$

これより，ふたつの係数は大きさが等しく符号が逆であることが分かります。この関係を仮定の式に戻します。

$$
|0, 0\rangle = A |+\frac{1}{2}, -\frac{1}{2}\rangle - A |-\frac{1}{2}, +\frac{1}{2}\rangle
$$

状態の規格化条件を課します。

$$
\langle 0, 0|0, 0\rangle = |A|^2 + |-A|^2 = 2|A|^2 = 1 \implies |A|^2 = \frac{1}{2}
$$

問題の指示より， $|+\frac{1}{2}, -\frac{1}{2}\rangle$ の係数である $A$ は正の実数とするため， $A = \frac{1}{\sqrt{2}}$ と定まります。自動的に $B = -\frac{1}{\sqrt{2}}$ となります。

$$
|0, 0\rangle = \frac{1}{\sqrt{2}} |+\frac{1}{2}, -\frac{1}{2}\rangle - \frac{1}{\sqrt{2}} |-\frac{1}{2}, +\frac{1}{2}\rangle
$$

このように，全角運動量が 1 の三重項状態は符号がプラスで対称になり， 0 の一重項状態は符号がマイナスで反対称になる組み合わせが導かれました。
