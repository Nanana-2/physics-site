## このサイトは何者？

ご覧いただきありがとうございます！本サイトは**大学定期試験〜大学院入試レベルの物理の問題とその解説をまとめた問題集**です。

スマートフォンやタブレットからの閲覧にも対応しててとっても便利……なはず？ですがパソコンで見ることを推奨します。式が長いので。

---


今日（こんにち），物理の問題の解説なんてAIに聞けばいくらでも出てきますし，作問だってAIにやらせることができます。前提として**このサイトの問題・解説もその殆どがAIによって作られています。**かがくのちからってすげー！

でして，AIはどんな質問にも答えてくれますが，一方で物事を体系的に知ることに関してはまだ課題があると思っています。これはプロンプトを入力する側が体系的な知識を持っていないことに依るものです。

そこでこのサイトでは，自分自身が院試勉強をちょっと頑張った経験をもとに，**学部4年の夏（院試）までに最低限ここら辺は知っていた**という目安を，問題という形で体系的にまとめ，提供します。

院試を控えている方にとっては対策の目安に，現在進行形で大学物理を学んでいる方には「この分野にはどんな話題があるのか」を体系的に知る一つのきっかけになれば嬉しいです。

……などと適当言ってますけど，**実態はAIと遊んでみようの思い付きで作ったものです。**遊んでいたら楽しくなって院試にはまず出ない内容まで紛れ込んでいるのもご愛嬌！そんなこともあります。

**ただし原則として自分自身が学部の時に知っていたこと以上の話題は入れない**ようにしています。ですので一学生が学部の間に学ぶことのサンプルケースとして見ていただければ幸いです。


## 重要な約束と推奨される使い方

このサイトで一番重要なこと，それは**このサイトの問題・解説の殆どはAIが作ったもので，それを見て確かめているのはただの院生**ということです。あまりに大事なので**このページだけは純度100％自分の手で書いています。**

ですのでこのサイトにおける内容の信憑性は**「ないよりはマシ」**というのが一番表現として正確です。**責任は，取れません！**

もちろん作る側としては（ざっくり）見てはいますし扱っている問題のレベルも精々学部の域を出ないので，致命的な間違いはそこまでないと踏んでいますが，あくまで「世の中にはこういう話があるんだな」と眺めてもらう程度の距離感で使ってもらうことを推奨しています。

もし間違いだったり気になる点があればXのDMにでも（優しく）送っていただけますと目を通す，かも。





## 収録問題・分野 { #fields }

主に大学物理の中でも**物性物理を学ぶ上で基礎となる分野**（力学・電磁気学・熱力学・量子力学・統計力学・物性物理）について，**三段階の難易度のタグ付け**（基本・標準・発展）がされています。目安としては

* **基本**…定期試験に出てくる程度
* **標準**…院試におけるベーシックな問題程度
* **発展**…やや発展的トピック。過去問ではたまに見かけたから知っておいて損はない程度のものから試験ではまずでないトピックまで
