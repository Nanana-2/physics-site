# ランダウ準位

## 問題
一様な磁場中を運動する二次元の荷電粒子は，離散的なエネルギー準位をとる。この準位をランダウ準位と呼び，各準位には多数の状態が縮退している。

$z$ 軸の正方向に一様な磁場 $\boldsymbol{B} = (0, 0, B_0)$ がかかっている。電荷 $q>0$，質量 $m$ の粒子が，$x$ 方向の長さ $L_x$，$y$ 方向の長さ $L_y$，面積 $S = L_xL_y$ の長方形領域内を運動する。$x$ 方向には周期境界条件 $\psi(x + L_x, y) = \psi(x, y)$ を課す。

ランダウゲージ $\boldsymbol{A} = (-B_0 y, 0, 0)$ を用いたときの二次元ハミルトニアンは以下の通りである。

$$
\hat{H} = \frac{\hat{p}_y^2}{2m} + \frac{1}{2m} (\hat{p}_x + qB_0 y)^2
$$

前問の結果より，このハミルトニアンの固有値問題は，サイクロトロン角振動数 $\omega_c = \frac{qB_0}{m}$ を持ち，中心位置が $y_0 = -\frac{p_x}{qB_0}$ にずれた1次元調和振動子の方程式に帰着される。以下の問いに答えよ。

(1) 1次元調和振動子の固有値の公式を利用して，荷電粒子がとり得るエネルギー固有値 $E_n$ の式を，非負の整数である量子数 $n = 0, 1, 2, \dots$ を用いて表せ。

(2) $x$ 方向の周期境界条件から，運動量の固有値 $p_x$ が満たすべき量子化条件を求めよ。 $p_x$ を，整数 $k$ および $L_x$ ， $\hbar$ を用いて表せ。

(3) 振動の中心位置 $y_0$ が，粒子の存在する実際の $y$ 方向の領域 $0 \leq y \leq L_y$ の中に入っていなければならないという条件を考える。この条件と (2) の結果を組み合わせることで，整数 $k$ が取り得る値の範囲を不等式で示せ。

(4) 1つのランダウ準位 $E_n$ に含まれる，異なる $k$ の状態の総数を数え上げることで，縮退度 $N$ を求めよ。また，その縮退度 $N$ が，領域を貫く全磁通 $\Phi = B_0 S$ を磁束量子 $\Phi_0 = \frac{2\pi\hbar}{q}$ で割り算した値 $\Phi / \Phi_0$ に一致することを示せ。

---

## 解説

### (1)

変形された方程式は，中心位置が $y_0$ に移動しているだけで，微分演算子と自乗の係数がなす構造は通常の1次元調和振動子と全く同じです。

$$
\left[ \frac{\hat{p}_y^2}{2m} + \frac{1}{2} m \omega_c^2 (y - y_0)^2 \right] \chi(y) = E \chi(y)
$$

したがって，エネルギー固有値は調和振動子のエネルギー量子化の公式がそのまま適用されます。

$$
E_n = \hbar \omega_c \left( n + \frac{1}{2} \right) \quad (n = 0, \, 1, \, 2, \, \dots)
$$

この離散的なエネルギー準位 $E_n$ がランダウ準位です。エネルギーの間隔は常に $\hbar \omega_c$ で一定となります。

### (2)

$x$ 方向の波動関数は $e^{i p_x x / \hbar}$ という平面波の形を持っています。これに周期境界条件 $\psi(x + L_x, y) = \psi(x, y)$ を適用します。

$$
e^{i p_x (x + L_x) / \hbar} = e^{i p_x x / \hbar} \implies e^{i p_x L_x / \hbar} = 1
$$

指数関数の周期性から，指数の肩にある値が $2\pi$ の整数倍にならなければなりません。整数を $k$ と置きます。

$$
\frac{p_x L_x}{\hbar} = 2\pi k \implies p_x = \frac{2\pi\hbar}{L_x} k \quad (k = 0, \, \pm 1, \, \pm 2, \, \dots)
$$

これが $x$ 方向の運動量に課される量子化条件です。

### (3)

振動の中心位置の数式 $y_0 = -\frac{p_x}{qB_0}$ に， (2) で得られた $p_x$ の制限を代入します。

$$
y_0 = -\frac{2\pi\hbar}{qB_0 L_x} k
$$

中心位置 $y_0$ は $0 \leq y_0 \leq L_y$ を満たす必要があります。$q>0$，$B_0>0$ のとき，整数 $k$ は 0 または負の整数です。そこで $\kappa \geq 0$ として $k = -\kappa$ と置き，不等式を整理します。

$$
0 \leq \frac{2\pi\hbar}{qB_0 L_x} \kappa \leq L_y
$$

中央の辺にある係数を払うために，各辺に $\frac{qB_0 L_x}{2\pi\hbar}$ を掛け算します。

$$
0 \leq \kappa \leq \frac{qB_0 L_x L_y}{2\pi\hbar}
$$

もとの整数 $k$ の表記に戻すと，取り得る範囲は以下の不等式で表されます。

$$
-\frac{qB_0 L_x L_y}{2\pi\hbar} \leq k \leq 0
$$

### (4)

(3) で求めた範囲に含まれる整数 $k$ の個数が，そのまま同じエネルギー $E_n$ を共有する状態の数，すなわち縮退度 $N$ に対応します。上限の 0 から下限の値を引き算し，境界の分として 1 を加えます。領域の面積 $S = L_x L_y$ を代入します。

$$
N = 0 - \left( -\frac{qB_0 L_x L_y}{2\pi\hbar} \right) = \frac{qB_0 S}{2\pi\hbar}
$$

この数式を問題文の指示通りに変形します。全磁通が $\Phi = B_0 S$ であり，磁束量子が $\Phi_0 = \frac{2\pi\hbar}{q}$ であるため，これらを縮退度の式に当てはめます。

$$
N = \frac{B_0 S}{\frac{2\pi\hbar}{q}} = \frac{\Phi}{\Phi_0}
$$

計算結果より，1つのランダウ準位に属する状態の総数は，そのサンプルの面内を通過する全磁束が，量子力学的な最小単位である磁束量子をいくつ含むかという比率に一致することが示されました。
