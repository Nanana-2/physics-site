# 昇降演算子と角運動量固有値

## 問題
角運動量演算子の大きさを表す $\hat{L}^2$ と，その1成分である $\hat{L}_z$ は互いに可換であり，共通の固有状態を持つ。

角運動量の全平方演算子 $\hat{L}^2 = \hat{L}_x^2 + \hat{L}_y^2 + \hat{L}_z^2$ と $z$ 成分 $\hat{L}_z$ の同時固有状態を $|l, m\rangle$ とし，それぞれの固有値方程式を以下のように置く。

$$
\hat{L}^2 |l, m\rangle = \lambda |l, m\rangle, \quad \hat{L}_z |l, m\rangle = m\hbar |l, m\rangle
$$

ここで，角運動量の昇降演算子を $\hat{L}_+ = \hat{L}_x + i\hat{L}_y$ ， $\hat{L}_- = \hat{L}_x - i\hat{L}_y$ と定義する。成分間の交換関係は $[\hat{L}_x, \hat{L}_y] = i\hbar \hat{L}_z$ ， $[\hat{L}_z, \hat{L}_x] = i\hbar \hat{L}_y$ である。以下の問いに答えよ。

(1) 交換関係から導かれる関係式 $[\hat{L}_z, \hat{L}_\pm] = \pm\hbar \hat{L}_\pm$ を用いて，状態 $\hat{L}_\pm |l, m\rangle$ に対する $\hat{L}_z$ の固有値方程式を計算し，これらの演算子が $m$ の値を 1 ずつ上下させる性質を持つことを示せ。

(2) 積の演算子 $\hat{L}_- \hat{L}_+$ および $\hat{L}_+ \hat{L}_-$ を展開し， $\hat{L}^2$ と $\hat{L}_z$ を用いてそれぞれ表せ。

(3) 状態 $\hat{L}_+ |l, m_{\text{max}}\rangle = 0$ となる最大の $z$ 成分量子数を $m_{\text{max}} = l$ と置く。 (2) の結果を利用して， $\hat{L}^2$ の固有値 $\lambda$ が $l$ を用いて $\lambda = l(l+1)\hbar^2$ と表されることを示せ。また，同様に $\hat{L}_- |l, m_{\text{min}}\rangle = 0$ となる最小の値から， $m$ の取り得る範囲が $-l \le m \le l$ となることを導け。

---

## 解説

### (1)

状態 $\hat{L}_\pm |l, m\rangle$ に左から $\hat{L}_z$ を作用させます。提示された交換関係より $\hat{L}_z \hat{L}_\pm = \hat{L}_\pm \hat{L}_z \pm \hbar \hat{L}_\pm$ と順序を入れ替えることができます。

$$
\hat{L}_z \left( \hat{L}_\pm |l, m\rangle \right) = (\hat{L}_z \hat{L}_\pm) |l, m\rangle = (\hat{L}_\pm \hat{L}_z \pm \hbar \hat{L}_\pm) |l, m\rangle
$$

右側の状態に $\hat{L}_z$ を作用させて固有値を前に出します。

$$
\hat{L}_\pm (m\hbar |l, m\rangle) \pm \hbar \hat{L}_\pm |l, m\rangle = (m \pm 1)\hbar \left( \hat{L}_\pm |l, m\rangle \right)
$$

これにより，新しく作られた状態の $\hat{L}_z$ に対する固有値は $(m \pm 1)\hbar$ になることが分かります。 $\hat{L}_+$ は $m$ を 1 つ上げ， $\hat{L}_-$ は $m$ を 1 つ下げる演算子です。

### (2)

定義に従って掛け算を行います。成分同士が非可換であることに注意します。

$$
\hat{L}_- \hat{L}_+ = (\hat{L}_x - i\hat{L}_y)(\hat{L}_x + i\hat{L}_y) = \hat{L}_x^2 + i\hat{L}_x \hat{L}_y - i\hat{L}_y \hat{L}_x + \hat{L}_y^2
$$

$$
= \hat{L}_x^2 + \hat{L}_y^2 + i(\hat{L}_x \hat{L}_y - \hat{L}_y \hat{L}_x) = \hat{L}_x^2 + \hat{L}_y^2 + i[\hat{L}_x, \hat{L}_y]
$$

ここに $\hat{L}_x^2 + \hat{L}_y^2 = \hat{L}^2 - \hat{L}_z^2$ および交換関係 $[\hat{L}_x, \hat{L}_y] = i\hbar \hat{L}_z$ を代入します。

$$
\hat{L}_- \hat{L}_+ = \hat{L}^2 - \hat{L}_z^2 + i(i\hbar \hat{L}_z) = \hat{L}^2 - \hat{L}_z^2 - \hbar \hat{L}_z
$$

符号を反転させて同様に計算すると，逆の順序の積が得られます。

$$
\hat{L}_+ \hat{L}_- = \hat{L}^2 - \hat{L}_z^2 + \hbar \hat{L}_z
$$

### (3)

これ以上 $m$ の値を上げることができない最大の状態 $|l, l\rangle$ に $\hat{L}_+$ を作用させると 0 になります。

$$
\hat{L}_+ |l, l\rangle = 0
$$

この式の左から $\hat{L}_-$ を掛け算します。右辺は 0 のままです。

$$
\hat{L}_- \hat{L}_+ |l, l\rangle = 0
$$

(2) で求めた関係式を代入します。

$$
(\hat{L}^2 - \hat{L}_z^2 - \hbar \hat{L}_z) |l, l\rangle = 0
$$

固有値方程式をそれぞれ適用します。

$$
\lambda |l, l\rangle - (l\hbar)^2 |l, l\rangle - \hbar(l\hbar) |l, l\rangle = 0
$$

$$
\left( \lambda - l^2\hbar^2 - l\hbar^2 \right) |l, l\rangle = 0
$$

状態ベクトルは 0 ではないため，係数の括弧の中身が 0 になります。これより $\hat{L}^2$ の固有値が決定します。

$$
\lambda = l(l+1)\hbar^2
$$

次に，これ以上 $m$ の値を下げることができない最小の状態 $|l, m_{\text{min}}\rangle$ に対して $\hat{L}_- |l, m_{\text{min}}\rangle = 0$ と置き，左から $\hat{L}_+$ を掛け算します。

$$
\hat{L}_+ \hat{L}_- |l, m_{\text{min}}\rangle = 0 \implies (\hat{L}^2 - \hat{L}_z^2 + \hbar \hat{L}_z) |l, m_{\text{min}}\rangle = 0
$$

すでに求めた $\lambda = l(l+1)\hbar^2$ と，固有値 $m_{\text{min}}\hbar$ を代入します。

$$
l(l+1)\hbar^2 - m_{\text{min}}^2\hbar^2 + m_{\text{min}}\hbar^2 = 0 \implies l^2 + l - m_{\text{min}}^2 + m_{\text{min}} = 0
$$

因数分解して整理します。

$$
(l + m_{\text{min}})(l - m_{\text{min}}) + (l + m_{\text{min}}) = 0 \implies (l + m_{\text{min}})(l - m_{\text{min}} + 1) = 0
$$

$m_{\text{min}}$ は最大値 $l$ 以下の値をとるため，第2項の括弧が 0 になることはありません。したがって第1項の括弧が 0 になり， $m_{\text{min}} = -l$ が定まります。

昇降演算子の作用によって $m$ は 1 ずつ変化するため，最大値から最小値までの間には整数回のステップが存在します。これにより， $m$ の範囲は $-l$ から $l$ までの等間隔な領域に制限されることが示されました。
