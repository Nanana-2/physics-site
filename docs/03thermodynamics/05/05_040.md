# 熱力学ポテンシャルの自然変数

## 問題
熱力学体系を記述する4つの熱力学ポテンシャルについて，それぞれの全微分形式を導出し，微視的な状態の変化に連動する自然な変数の組み合わせを整理する。
熱力学第一法則の微分形式 $dU = T dS - P dV$ を基本式として，以下の問いに答えよ。

(1) エンタルピー $H = U + PV$ の全微分 $dH$ を計算し， $dS$ と $dP$ を用いた形式で表せ。また，この式からエンタルピーの自然な変数が $S$ と $P$ であることを示せ。

(2) ヘルムホルツ自由エネルギー $F = U - TS$ の全微分 $dF$ を計算し， $dT$ と $dV$ を用いた形式で表せ。また，この式からヘルムホルツ自由エネルギーの自然な変数が $T$ と $V$ であることを示せ。

(3) ギブズ自由エネルギー $G = H - TS$ の全微分 $dG$ を計算し， $dT$ と $dP$ を用いた形式で表せ。また，この式からギブズ自由エネルギーの自然な変数が $T$ と $P$ であることを示せ。

---

## 解説

### (1)
エンタルピーの定義式の両辺について全微分をとります。積の微分公式を適用すると次のようになります。

$$
dH = dU + P dV + V dP
$$

この式に，内部エネルギーの全微分形式 $dU = T dS - P dV$ を代入します。

$$
dH = (T dS - P dV) + P dV + V dP
$$

右辺の $P dV$ の項が互いに相殺されます。式を整理すると次のようになります。

$$
dH = T dS + V dP
$$

この全微分形式において，右辺の微小変化量は $dS$ と $dP$ です。したがって，エンタルピーの独立変数としてエントロピー $S$ と圧力 $P$ を選択したとき，その係数が温度 $T$ と体積 $V$ というシンプルな状態量だけで表現されるため，エンタルピーの自然な変数は $S$ と $P$ になります。

### (2)
ヘルムホルツ自由エネルギーの定義式の両辺について全微分をとります。積の微分公式を適用すると次のようになります。

$$
dF = dU - T dS - S dT
$$

この式に，基本式である $dU = T dS - P dV$ を代入します。

$$
dF = (T dS - P dV) - T dS - S dT
$$

右辺の $T dS$ の項が互いに相殺されます。式を整理すると次のようになります。

$$
dF = -P dV - S dT
$$

この全微分形式において，右辺の微小変化量は $dV$ と $dT$ です。したがって，ヘルムホルツ自由エネルギーの独立変数として体積 $V$ と温度 $T$ を選択したとき，その係数が圧力 $P$ とエントロピー $S$ のみで簡潔に表現されるため，ヘルムホルツ自由エネルギーの自然な変数は $T$ と $V$ になります。

### (3)
ギブズ自由エネルギーの定義式の両辺について全微分をとります。積の微分公式を適用すると次のようになります。

$$
dG = dH - T dS - S dT
$$

この式に，(1) で導出したエンタルピーの全微分形式 $dH = T dS + V dP$ を代入します。

$$
dG = (T dS + V dP) - T dS - S dT
$$

右辺の $T dS$ の項が互いに相殺されます。式を整理すると次のようになります。

$$
dG = V dP - S dT
$$

この全微分形式において，右辺の微小変化量は $dP$ と $dT$ です。したがって，ギブズ自由エネルギーの独立変数として圧力 $P$ と温度 $T$ を選択したとき，その係数が体積 $V$ とエントロピー $S$ のみで表現されるため，ギブズ自由エネルギーの自然な変数は $T$ と $P$ になります。
