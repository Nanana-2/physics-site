# ウィグナー分布と伏見関数の比較

## 問題
量子状態の位相空間プロファイルを調べる代表的な手法として，ウィグナー分布関数 $W(x, p)$ と伏見関数 $Q(x, p)$ が挙げられる。伏見関数はウィグナー分布関数を不確定性原理の最小限界の大きさを持つガウス関数で平滑化したものに等しい。ここでは，調和振動子の第1励起状態を具体的な共通の対象として選び，両者の関数形の違いを計算によって比較する。

質量 $m$ ，角振動数 $\omega$ の1次元調和振動子を考える。系の第1励起状態の波動関数は，代表的な長さのスケール $x_0 = \sqrt{\frac{\hbar}{m\omega}}$ を用いて以下のように与えられる。

$$
\psi(x) = \left( \frac{4}{\pi x_0^6} \right)^{1/4} x \exp\left( -\frac{x^2}{2x_0^2} \right)
$$

この状態に対するウィグナー分布関数 $W(x, p)$ は，代表的な運動量のスケール $p_0 = \frac{\hbar}{x_0} = \sqrt{m\omega\hbar}$ と，無次元化された位相空間の半径の2乗 $R^2 = \frac{x^2}{x_0^2} + \frac{p^2}{p_0^2}$ を用いて，すでに以下のように計算されているものとする。

$$
W(x, p) = \frac{1}{\pi\hbar} \left( 2R^2 - 1 \right) \exp\left( -R^2 \right)
$$

以下の問いに答えよ。

(1) 提示されたウィグナー分布関数 $W(x, p)$ について，位相空間の原点 $x = 0, p = 0$ における値を求めよ。また，この関数が負の値をとる位相空間上の領域を $R$ の不等式で表せ。

(2) 伏見関数 $Q(x, p)$ は，上記の $W(x, p)$ に対して以下の積分変換を施すことで得られる。

$$
Q(x, p) = \frac{1}{\pi\hbar} \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} \exp\left( - \frac{(x - x')^2}{x_0^2} - \frac{(p - p')^2}{p_0^2} \right) W(x', p') dx' dp'
$$

この積分を実行すると，伏見関数の具体的な形は以下のようになる。

$$
Q(x, p) = \frac{1}{2\pi\hbar} R^2 \exp\left( -\frac{1}{2}R^2 \right)
$$

この伏見関数 $Q(x, p)$ について，位相空間上の任意の点における値が常に 0 以上であることを式から示せ。また，原点 $x = 0, p = 0$ における値を求めよ。

(3) (1) と (2) の結果を比較せよ。同じ量子状態を対象にしているにもかかわらず，原点付近においてウィグナー分布関数には負の谷が現れ，伏見関数では値が 0 となって負の領域が完全に消失する物理的な仕組みを，ガウスフィルターによる平滑化の効果と関連付けて説明せよ。

---

## 解説

### (1)

提示されたウィグナー分布関数の式に，原点条件 $x = 0, p = 0$ を代入します。このとき無次元半径の2乗は $R^2 = 0$ となります。

$$
W(0, 0) = \frac{1}{\pi\hbar} \left( 2 \cdot 0 - 1 \right) \exp\left( 0 \right) = -\frac{1}{\pi\hbar}
$$

原点における値は負の実数となることがわかります。

次に，この関数が負の値をとる条件を調べます。定数係数および指数関数の部分は常に正であるため，括弧の内部の符号が負になれば全体も負になります。

$$
2R^2 - 1 < 0 \implies R^2 < \frac{1}{2} \implies R < \frac{1}{\sqrt{2}}
$$

したがって，ウィグナー分布関数は，位相空間の原点を中心とする半径 $\frac{1}{\sqrt{2}}$ 未満の円盤状の領域において，一様に負の値をとることが示されます。

### (2)

与えられた伏見関数の式に注目します。

$$
Q(x, p) = \frac{1}{2\pi\hbar} R^2 \exp\left( -\frac{1}{2}R^2 \right)
$$

定数係数は正であり，指数関数部分も常に正です。残る変数部分は座標の2乗の和である $R^2 \ge 0$ です。すべての因数が 0 以上であるため，位相空間の全域において以下の不等式が満たされます。

$$
Q(x, p) \ge 0
$$

次に，原点 $x = 0, p = 0$ を代入します。 $R^2 = 0$ となるため，式全体に掛け算されて結果は 0 になります。

$$
Q(0, 0) = \frac{1}{2\pi\hbar} \cdot 0 \cdot \exp\left( 0 \right) = 0
$$

伏見関数は原点において負に沈み込むことなく，ちょうど 0 という最小値をとることがわかります。

### (3)

調和振動子の第1励起状態は，エネルギーの節が存在する量子力学的な波の性質を強く持っています。ウィグナー分布関数はこの微視的な波の干渉を敏感に捉えるため，(1) で見たように原点近傍の狭い領域にシャープな負の谷を形成し，その周囲に正のドーナツ状の山が取り囲むという，激しい凹凸を持ったプロファイルを描きます。

これに対して伏見関数を作る積分変換は，位相空間の各点に対してプランク定数程度の広がりを持つ2次元のガウスフィルターを掛け，周囲の値を足し合わせる平滑化の操作に相当します。

原点付近に注目すると，ウィグナー分布関数が持つ中心の負の谷と，そのすぐ外側にある正のドーナツ状の山の値が，ガウスフィルターによる平均化の過程で互いに相殺し合います。この粗視化による相殺の効果が非常に精密に働くため，中心の負の底が引き上げられ，最終的な伏見関数のプロファイルでは原点における値がぴったり 0 に平ら化されます。これにより，不確定性原理のスケールよりも細かな量子干渉の構造が滑らかに削ぎ落とされ，全域で 0 以上という古典的な確率に近い滑らかなプロファイルへと変形することになります。
