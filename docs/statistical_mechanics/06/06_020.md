# 占有数表示

## 問題
互いに相互作用しない $N$ 個の同種粒子（ボース粒子またはフェルミ粒子）からなる量子力学的な系を考える。この系における単一粒子の独立な量子状態（準位）をエネルギーの低い順に $k = 1, 2, 3, \dots$ とラベルづけし，それぞれの準位のエネルギーを $\epsilon_1, \epsilon_2, \epsilon_3, \dots$ とする。

粒子が区別できない同種粒子系では，個々の粒子に番号をつけて状態を指定する代わりに，各準位 $k$ に何個の粒子が配置されているかという**占有数** $n_k$ の順序対 $(n_1, n_2, n_3, \dots)$ を用いて系全体の微視的状態を指定する手法が極めて有効である。これを占有数表示と呼ぶ。以下の問いに答えよ。

(1) 粒子がボース粒子（ボソン）である場合を考える。ある特定の準位 $k$ に入ることができる粒子の個数（占有数 $n_k$）として許される値の範囲を，非負の整数を用いて示せ。また，全粒子数が $N$ で固定されているカノニカル集団において，各準位の占有数の総和が満たすべき拘束条件の式を記述せよ。

(2) 粒子がフェルミ粒子（フェルミオン）である場合を考える。パウリの排他原理を考慮したとき，ある特定の準位 $k$ の占有数 $n_k$ として許される値をすべて挙げよ。

(3) 3つの単一粒子準位 $k = 1, 2, 3$ のみが存在し，そこに合計で 2 個の同種粒子（ $N=2$ ）を配置する具体的な状況を考える。

* 粒子がボース粒子である場合
* 粒子がフェルミ粒子である場合

それぞれのケースについて，系全体としてとり得るすべての微視的状態を占有数の組 $(n_1, n_2, n_3)$ の形式で具体的にすべて列挙せよ。また，状態の総数（全状態数）が両者でどのように異なるかを比較せよ。

---

## 解説

### (1)

ボース粒子は，波動関数が座標の入れ替えに対して対称であるという性質を持ち，複数の粒子が同じ量子状態を共有することに対して何の制限もありません。

したがって，1つの準位 $k$ を占有できる粒子の数 $n_k$ は，全く粒子が入っていない状態からすべての粒子が凝縮する状態まで，以下の任意の非負整数をとることができます。

$$
n_k = 0, 1, 2, 3, \dots
$$

また，系全体の全粒子数が $N$ 個に固定されているカノニカル集団においては，すべての準位の占有数を足し合わせた合計が必ず $N$ に一致しなければならないため，以下の拘束条件が課されます。

$$
\sum_{k} n_k = N
$$

### (2)

フェルミ粒子は，波動関数が座標の入れ替えに対して反対称であるという性質を持ち，パウリの排他原理が厳密に適用されます。前問で示された通り，同一の単一粒子状態に2つ以上の粒子が同時に存在することは不可能です。

したがって，各準位 $k$ の占有数 $n_k$ は，粒子が存在しないか，あるいは最大でも1個だけ存在するかの2通りの値しかとることができません。

$$
n_k = 0, 1
$$

### (3)

3つの準位に対して 2 個の粒子を配置するすべての組み合わせを，書き漏らしのないように探索します。常に $n_1 + n_2 + n_3 = 2$ の条件が満たされている必要があります。

**ボース粒子のケース**
ボース粒子では同一の準位に 2 個の粒子が同時に入る（ $n_k = 2$ ）ことが許されます。条件を満たす状態を列挙すると，以下の **6通り** となります。

* 1つの準位に2粒子が固まって入る状態：
  $(2, 0, 0), \ (0, 2, 0), \ (0, 0, 2)$
* 異なる2つの準位に1粒子ずつ分散して入る状態：
  $(1, 1, 0), \ (1, 0, 1), \ (0, 1, 1)$



**フェルミ粒子のケース**

フェルミ粒子では $n_k$ として $0$ または $1$ しか許されないため，同一準位に 2 個の粒子が入る状態はすべて排除されます。必ず異なる2つの準位を選んで1文字ずつ配置することになるため，とり得る状態は以下の **3通り** に限定されます。

* $(1, 1, 0), \ (1, 0, 1), \ (0, 1, 1)$



**全状態数の比較**

ボース粒子では全状態数が 6 通りであるのに対し，フェルミ粒子ではパウリの排他原理による禁止則が働くため，全状態数が 3 通りへと減少します。このように，粒子の持つ波動関数の対称性の違いは，微視的な状態の数え上げ（状態密度）に直接影響を与え，これがマクロな熱力学的な振る舞いや分布関数の違いを生み出す根本的な原因となります。
