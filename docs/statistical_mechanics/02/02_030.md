# 二つの系の熱平衡条件

## 問題
完全に孤立した大きな容器の中に，熱を伝えることができる隔壁で仕切られた2つの部分系（系1と系2）が存在している。全体の全エネルギーは $E_{\mathrm{tot}}$ で一定に保たれており，系1のエネルギーを $E_1$，系2のエネルギーを $E_2$ とすると，常に $E_1 + E_2 = E_{\mathrm{tot}}$ が成り立つ。各系がエネルギー $E_i$ を持つときの微視的状態数を $\Omega_i(E_i)$ （$i=1, 2$）とするとき，以下の問いに答えよ。

(1) 全体の微視的状態数 $\Omega_{\mathrm{tot}}(E_1)$ が最大になるとき，全体のボルツマンエントロピー $S_{\mathrm{tot}} = S_1(E_1) + S_2(E_2)$ も最大となる。$S_{\mathrm{tot}}$ が最大値をとるための必要条件を，$S_1$ および $S_2$ のエネルギー微分を用いて表せ。
(2) エネルギーの微小な変化量 $dE_1$ と $dE_2$ の間に成り立つ関係式を示し，(1)の条件を各系の統計力学的な逆温度 $\beta_i = \frac{\partial \log \Omega_i}{\partial E_i}$ を用いた簡潔な関係式に書き換えよ。
(3) この全エントロピー最大化のプロセスが，熱力学における「熱が温度の高い方から低い方へ流れ，最終的に熱平衡（等温）に達する」という現象にどのように対応しているか，物理的な論理の流れを説明せよ。

---

## 解説

### (1)

全体のボルツマンエントロピー $S_{\mathrm{tot}}$ が最大値をとるとき，$E_1$ に関する1階微分は 0 にならなければなりません。したがって，求める必要条件は以下のようになります。

$$
\frac{dS_{\mathrm{tot}}}{dE_1} = \frac{d}{dE_1} \left[ S_1(E_1) + S_2(E_2) \right] = 0
$$

合成関数の微分法（連鎖律）を適用すると，次の式が得られます。

$$
\frac{dS_1(E_1)}{dE_1} + \frac{dS_2(E_2)}{dE_2} \frac{dE_2}{dE_1} = 0
$$

### (2)

全体のエネルギーが一定（$E_1 + E_2 = E_{\mathrm{tot}}$）であるため，その全微分をとると $dE_1 + dE_2 = 0$ となり，変化量の間の関係式は以下のようになります。

$$
\frac{dE_2}{dE_1} = -1
$$

これを (1) で得られた式に代入します。

$$
\frac{dS_1(E_1)}{dE_1} - \frac{dS_2(E_2)}{dE_2} = 0 \quad \implies \quad \frac{dS_1(E_1)}{dE_1} = \frac{dS_2(E_2)}{dE_2}
$$

ボルツマンエントロピーの定義 $S_i = k_B \log \Omega_i$ を用いて両辺を $k_B$ で割ると，逆温度 $\beta_i = \frac{\partial \log \Omega_i}{\partial E_i}$ の定義そのものになります。したがって，求める関係式は以下の通りです。

$$
\beta_1 = \beta_2
$$

### (3)

熱平衡に達していない初期状態として，$\beta_1 > \beta_2$ （すなわち温度が $T_1 < T_2$）の状況を考えます。このとき，全エントロピーの微小変化は以下のように表せます。

$$
dS_{\mathrm{tot}} = \frac{dS_1}{dE_1} dE_1 + \frac{dS_2}{dE_2} dE_2 = k_B (\beta_1 dE_1 + \beta_2 dE_2)
$$

$dE_2 = -dE_1$ を代入して整理します。

$$
dS_{\mathrm{tot}} = k_B (\beta_1 - \beta_2) dE_1
$$

孤立系において自発的な変化はエントロピーが増加する方向（$dS_{\mathrm{tot}} > 0$）に進行するため，$\beta_1 - \beta_2 > 0$ であるならば，$dE_1 > 0$ でなければなりません。これは，温度の「高い」系2から温度の「低い」系1へとエネルギー（熱）が自発的に移動することを意味します。
このエネルギー移動に伴って $\beta_1$ と $\beta_2$ の差は縮まっていき，最終的に $\beta_1 = \beta_2$ （$T_1 = T_2$）となった時点でエントロピーが最大化され，マクロな熱の移動が停止する「熱平衡状態」へと至ります。
