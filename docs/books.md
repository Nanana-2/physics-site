# 参考文献一覧


## このページは何？

このページでは主にサイトを作るにあったって参考にしている……あるいは学部時代にお世話になった教科書諸々についてまとめています。

何度でも言いますがこのサイトをあてにするよりかは**参考文献の方がよっぽどあてにはなります。**一方，参考文献の妥当性についてはそこそこアテになる……と信じてます！つまりこのサイトで一番信頼できるページはここだと言いたい。情けない……

また本サイトはAmazonアソシエイトに参加しています。記事自体にアドセンスはつけていないので，良かったら以下の教科書だけでも！見てくれると！嬉しくなるかもしれません。
