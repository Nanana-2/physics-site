# エンタングルメントエントロピー

## 問題
合成系全体の量子状態が純粋状態であるとき，一方の部分系を部分トレースして得られる縮約密度行列のフォン・ノイマンエントロピーは，エンタングルメントエントロピーと呼ばれる。この量は，2つの部分系の間に存在する量子もつれの強さを定量的に評価するための指標となる。

2つの量子ビット系AとBからなる合成系を考える。全系の正規直交基底を $\{|00\rangle, |01\rangle, |10\rangle, |11\rangle\}$ とする。実数のパラメータ $\theta$ を用いて，全系の純粋状態 $|\Psi(\theta)\rangle$ が以下のように与えられている。

$$
|\Psi(\theta)\rangle = \cos\theta |0_{\mathrm{A}} 0_{\mathrm{B}}\rangle + \sin\theta |1_{\mathrm{A}} 1_{\mathrm{B}}\rangle
$$

以下の問いに答えよ。

(1) 状態 $|\Psi(\theta)\rangle$ に対する合成系の全密度行列 $\rho_{\mathrm{tot}}(\theta) = |\Psi(\theta)\rangle\langle\Psi(\theta)|$ を展開せよ。

さらに，系Bに関する部分トレースを実行し，系Aの縮約密度行列 $\rho_{\mathrm{A}}(\theta) = \mathrm{Tr}_{\mathrm{B}}(\rho_{\mathrm{tot}}(\theta))$ を $|0_{\mathrm{A}}\rangle, |1_{\mathrm{A}}\rangle$ を基底とする行列形式で求めよ。

(2) (1) で得られた縮約密度行列 $\rho_{\mathrm{A}}(\theta)$ を用いて，系Aのエンタングルメントエントロピー $E(\theta) = -k_B \mathrm{Tr}(\rho_{\mathrm{A}}(\theta) \ln \rho_{\mathrm{A}}(\theta))$ を $\theta$ の関数として式で表せ。

(3) 指定された3つの $\theta$ について $E(\theta)$ を求め，最大値と最小値を与える状態を答えよ。

* $\theta = 0$
* $\theta = \frac{\pi}{4}$
* $\theta = \frac{\pi}{2}$

---

## 解説

### (1)

状態 $|\Psi(\theta)\rangle$ のブラベクトルは以下の通りです。

$$
\langle\Psi(\theta)| = \cos\theta \langle 0_{\mathrm{A}} 0_{\mathrm{B}}| + \sin\theta \langle 1_{\mathrm{A}} 1_{\mathrm{B}}|
$$

全密度行列 $\rho_{\mathrm{tot}}(\theta) = |\Psi(\theta)\rangle\langle\Psi(\theta)|$ を分配法則に従って展開します。

$$
\rho_{\mathrm{tot}}(\theta) = \cos^2\theta |0_{\mathrm{A}} 0_{\mathrm{B}}\rangle\langle 0_{\mathrm{A}} 0_{\mathrm{B}}| + \cos\theta\sin\theta |0_{\mathrm{A}} 0_{\mathrm{B}}\rangle\langle 1_{\mathrm{A}} 1_{\mathrm{B}}| + \sin\theta\cos\theta |1_{\mathrm{A}} 1_{\mathrm{B}}\rangle\langle 0_{\mathrm{A}} 0_{\mathrm{B}}| + \sin^2\theta |1_{\mathrm{A}} 1_{\mathrm{B}}\rangle\langle 1_{\mathrm{A}} 1_{\mathrm{B}}|
$$

系Bの正規直交基底 $\{|0_{\mathrm{B}}\rangle, |1_{\mathrm{B}}\rangle\}$ を用いて部分トレースを行います。

$$
\rho_{\mathrm{A}}(\theta) = \langle 0_{\mathrm{B}}|\rho_{\mathrm{tot}}(\theta)|0_{\mathrm{B}}\rangle + \langle 1_{\mathrm{B}}|\rho_{\mathrm{tot}}(\theta)|1_{\mathrm{B}}\rangle
$$

系Bの直交条件により，非対角項はすべて 0 になり，対角項の系Bの成分が内積によって消去されます。

$$
\rho_{\mathrm{A}}(\theta) = \cos^2\theta |0_{\mathrm{A}}\rangle\langle 0_{\mathrm{A}}| + \sin^2\theta |1_{\mathrm{A}}\rangle\langle 1_{\mathrm{A}}|
$$

これを行列形式で書き表すと以下のようになります。

$$
\rho_{\mathrm{A}}(\theta) = \begin{pmatrix} \cos^2\theta & 0 \\ 0 & \sin^2\theta \end{pmatrix}
$$

### (2)

(1) で得られた縮約密度行列 $\rho_{\mathrm{A}}(\theta)$ はすでに対角化されているため，対角成分がそのまま固有値となります。固有値分布は以下の通りです。

$$
p_0 = \cos^2\theta, \quad p_1 = \sin^2\theta
$$

フォン・ノイマンエントロピーの定義式にこれらの固有値を代入します。

$$
E(\theta) = -k_B \left( \cos^2\theta \ln(\cos^2\theta) + \sin^2\theta \ln(\sin^2\theta) \right)
$$

これが求めるエンタングルメントエントロピーの式です。

### (3)

指定されたパラメータ $\theta$ の値を順に代入して計算を行います。ただし，必要に応じて極限公式 $\lim_{x \to 0} x \ln x = 0$ を使用します。

* **$\theta = 0$ のとき**
  
  $\cos^2 0 = 1, \sin^2 0 = 0$ となります。

  $$
  E(0) = -k_B \left( 1 \cdot \ln 1 + 0 \cdot \ln 0 \right) = -k_B (0 + 0) = 0
  $$
  
  **状態の説明**: 
  
  エントロピーが 0 であるため，系Aと系Bの間に量子もつれは一切存在しません。実際，元の状態は $|\Psi(0)\rangle = |0_{\mathrm{A}} 0_{\mathrm{B}}\rangle$ という単一の積状態に固定されています。

* **$\theta = \frac{\pi}{4}$ のとき**
  
  $\cos^2 \frac{\pi}{4} = \left(\frac{1}{\sqrt{2}}\right)^2 = \frac{1}{2}, \sin^2 \frac{\pi}{4} = \left(\frac{1}{\sqrt{2}}\right)^2 = \frac{1}{2}$ となります。
  
  $$
  E\left(\frac{\pi}{4}\right) = -k_B \left( \frac{1}{2} \ln \frac{1}{2} + \frac{1}{2} \ln \frac{1}{2} \right) = -k_B \ln \frac{1}{2} = k_B \ln 2
  $$
  
  **状態の説明**: 
  
  2次元量子系においてエントロピーが最大値の $k_B \ln 2$ に達しているため，系Aと系Bの間には最も強い量子もつれが形成されています。元の状態はベル状態 $|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$ に相当します。

* **$\theta = \frac{\pi}{2}$ のとき**
  
  $\cos^2 \frac{\pi}{2} = 0, \sin^2 \frac{\pi}{2} = 1$ となります。
  
  $$
  E\left(\frac{\pi}{2}\right) = -k_B \left( 0 \cdot \ln 0 + 1 \cdot \ln 1 \right) = -k_B (0 + 0) = 0
  $$
  
  **状態の説明**: 
  
  $\theta = 0$ のときと同様にエントロピーが 0 に減少しているため，系Aと系Bの間に量子もつれはありません。元の状態は基底が入れ替わっただけの単一の積状態 $|\Psi(\frac{\pi}{2})\rangle = |1_{\mathrm{A}} 1_{\mathrm{B}}\rangle$ となっています。
